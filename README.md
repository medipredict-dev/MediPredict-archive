# MediPredict
An AI-powered sports injury recovery prediction platform that helps athletes, coaches, and medical staff estimate accurate return-to-play timelines.​


## Project Description
MediPredict is a full stack web application aimed at assisting users in predicting
medical conditions using intelligent data-driven techniques.

This repository currently contains the initial project setup and documentation.
Development will begin in later phases.
